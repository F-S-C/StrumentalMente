Aiuto — Teoria

Ti trovi nella sezione teoria di StrumentalMente.

Puoi selezionare un livello tra quelli proposti cliccandovi sopra.

Per ogni livello ti saranno mostrati i suoi contenuti, a cui puoi accedere cliccandovi sopra.

Alla fine di un livello di teoria è presente un quiz che puoi utilizzare per mettere alla prova ciò che hai imparato.

Ecco un elenco di comandi che puoi eseguire:

  • Premi le frecce direzionali → e ← per sfogliare i contenuti, oppure i corrispondenti tasti Avanti e Indietro.
  • Premi CTRL + BACKSPACE oppure il pulsante Torna all'elenco per ritornare all'elenco degli argomenti.
  • Premi ESC per uscire dall'applicazione.
  • Premi CTRL + SPAZIO per mettere in play/pausa l'audio.
  • Premi CTRL + ALT + SPAZIO per stoppare e riavvolgere l'audio
  • Premi ALT+ H per ritornare alla Home Page.
  • Premi ALT+ P per accedere al tuo profilo.
  • Premi ALT +T per accedere alla teoria.
  • Premi ALT + S S per accedere alla sezione strumenti.
  • Premi ALT + S C per accedere alla teoria sulla chitarra.
  • Premi ALT + S B per accedere alla teoria sul basso.
  • Premi ALT + S P per accedere alla teoria sul piano.
  • Premi ALT + S SHIFT + B per accedere alla teoria sulla batteria.
  • Premi ALT + A C per accedere agli accordi della chitarra.
  • Premi ALT + A B per accedere agli accordi del basso.
  • Premi ALT + A P per accedere agli accordi del piano.
  • Premi ALT+Mper accedere alla mappa di navigazione del sito.
  • Premi ALT+Iper accedere ai crediti.
